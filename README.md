# 介绍
使用画布生成海报图片，根据配置参数方便快捷的批量作图。

![效果图](https://raw.githubusercontent.com/itzhenzichao/canvas-generate-picture/main/src/assets/image/effect.png)


# 用到的技术
* [![Vue][Vue.js]][Vue-url]
* [![Canvas][Canvas.com]][Canvas-url]
* [![Element][Element.com]][Element-url]


# 使用
### 下载依赖
      yarn install
### 运行
      yarn serve
### 打包
      yarn build
